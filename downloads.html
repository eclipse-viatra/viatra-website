VIATRA Downloads

VIATRA can be installed from Eclipse Marketplace.
You can simply drag the Marketplace button to an open Eclipse instance to start the installation.

Installing the current version

For manual installation, use provide the p2 repository url http://download.eclipse.org/viatra/updates/release/latest Installation instructions:

  • Choose Help -> Install New Software... from the menu bar and Add....
  • Select the VIATRA Query and Transformation SDK from the category VIATRA and complete the wizard by clicking the Next button until you can click Finish.
  • After a quick download and a restart of Eclipse, VIATRA is ready to use.

Development versions

The VIATRA project maintains different update sites to download from. The various sites contains versions of different stability and are subject of different amount of testing. CI and integration builds may contain breaking changes, while milestone and release builds should be stable for everyday use. As a general rule of thumb, you should use the release (or maybe the milestone) repository.

Warning! Only release builds are retained undefinitely, but other builds will be periodically removed.

Release composite
http://download.eclipse.org/viatra/updates/release - This site contains all releases of VIATRA. Note: It is not supported to combine features from various release repositories. When installing components from this site, ensure that everything selected comes from the same release. This can be problematic if a feature was removed, as old versions can still show up in the composite repository.
Milestone builds
http://download.eclipse.org/viatra/updates/milestone - This site contains milestone builds of upcoming VIATRA releases. A release cycle usually produces 3-4 milestone builds; these builds are also included in simrel milestones. Milestone builds are removed after the corresponding release build is finalized.
Integration builds
http://download.eclipse.org/viatra/updates/integration - This site contains weekly updated integration builds of upcoming VIATRA releases. These builds receive no specific testing; resort using these builds only if you need bleeding edge features or improvements.

Older releases

For information about previous releases, look at the VIATRA2 archive site and the EMF-IncQuery archive site.
