Add-ons
VIATRA provides various add-ons for a variaty of use cases

Model obfuscation

Model obfuscation enables you to share information and collaborate in new ways. It ensures that only model elements that you want to share can be viewed by certain people. Read more at https://wiki.eclipse.org/VIATRA/ModelObfuscator

Design space exploration

Design space exploration uses AI to aid creative engineering processes. With the top-notch resource of configuration generation, one can formulate a baseline, define self goals, and make any necessary design and space changes.
Read more at https://wiki.eclipse.org/VIATRA/DSE

IOT (complex event processing)

Today, the Internet of Things is the one of the fastest growing and innovative territories, making basic, everyday technologies simple to use for the average person. Yet, IoT challenges exist. These include, but aren’t limited to, on-the-fly filtering and high-level sensor data fusion for smart home gateways or smart sporting equipment. Luckily, VIATRA can help assist by providing advanced complex event processing frameworks such as Drools Fusion or VIATRA-CEP. With VIATRA, stream processors can be augmented or even replaced.

Model Validation Framework

Model Validation Framework allows you to define rules for well-formedness validation, which is an important development used to detect design flaws. This works in an incremental way, incorporating the Syntax driven modeling, guaranteeing the minimum amount of design flaws. Overall it’s a huge time saver.
Read more at https://wiki.eclipse.org/VIATRA/Addon/Validation

Viewers Framework

Viewers Framework is useful for view maintenance, a very fundamental component. This allows engineers to create user interface elements such as lists, trees, graphs and diagrams without programming in a general purpose language, such as Java. This is another time saver because those components typically require repetitive coding.
Read more at https://wiki.eclipse.org/VIATRA/Addon/VIATRA_Viewers
